# Family Market Website Documentation

## Overview
The Family Market website is an e-commerce platform designed to facilitate buying and selling of groceries and other family essentials. The website is user-friendly and responsive, ensuring a seamless shopping experience across different devices.

## Features
- **User Registration and Login**: New users can create an account and existing users can log in.
- **Product Listings**: Products are categorized for easy browsing.
- **Shopping Cart**: Users can add items to their cart and proceed to checkout.
- **Payment Integration**: Secure payment processing through various methods.
- **Order History**: Users can view their past orders.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript, React
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Hosting**: Heroku

## Setup Instructions
1. Clone the repository: `git clone https://github.com/nurdan55/family-market.git`
2. Navigate into the project directory: `cd family-market`
3. Install dependencies: `npm install`
4. Run the development server: `npm start`
